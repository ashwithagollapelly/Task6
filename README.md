# Task 6: Sales Trend Analysis Using SQL

This task analyzes monthly sales data using SQL aggregation functions to calculate monthly revenue, order volume, average order value, and month-over-month performance trends.

 Tools
- MySQL Workbench
- SQL (Aggregate & Date Functions)

Explanation

Created a new database and a table named orders to store sales records.

Inserted sample order data with fields such as order date, order ID, amount, and product ID.

Used SQL aggregate functions like SUM() and COUNT() to calculate monthly revenue and number of orders.

Grouped results by year and month to identify sales trends over time.

Calculated average order value and month-over-month percentage change to analyze performance growth.

Executed queries in MySQL Workbench and exported output screenshots for submission.
